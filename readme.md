### 简介
**BLAS（basic linear algebra subroutine）是一系列基本线性代数运算函数的接口（interface）标准，本项目根据标准接口实现新的BLAS版本。**

### 优化策略
* 分片执行
* OpenMP多线程
* 指令集优化SIMD
